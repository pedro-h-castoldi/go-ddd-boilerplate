# go-ddd-boilerplate
🇺🇸 A Go DDD boilerplate designed as a foundation for building APIs, background routines, and scripts using domain-driven design, and reusable project conventions.

⚠️ This project is currently under development.

🇧🇷 Um boilerplate em Go baseado em DDD, projetado como fundação para o desenvolvimento de APIs, rotinas em background e scripts, utilizando Domain-Driven Design e convenções reutilizáveis de projeto.

⚠️ Este projeto está atualmente em desenvolvimento.
